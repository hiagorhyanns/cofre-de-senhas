# Cofre de Senhas

Programa para Windows para salvar e organizar senhas.

## Download

[Baixar Cofre de Senhas para Windows](https://github.com/hiagorhyanns/cofre-de-senhas/releases/latest/download/senhas.exe)

Observação: o Windows pode exibir um aviso de segurança porque o programa ainda não possui assinatura digital.
